# sandbox-next-js-with-grpc

Sandbox for Next.js and gRPC with TypeScript.

![](https://i.gyazo.com/8768f29dd7de59ce2fd3c2daee8ebd66.png)

# Get started

Start mock server.

```
yarn dev:server
```

Start client with Next.js.

```
yarn dev:client
```

Generate code from Protocol Buffers.

```
yarn codegen
```
